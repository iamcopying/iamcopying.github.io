OpenClaw 和云端模型

发表于2026年3月2日 · 726个字

年初开始 OpenClaw 话题度只增不减，现在我们使用 OpenAI Codex 提供的免费额度体验下。

安装 OpenClaw: npm i -g openclaw

安装向导: openclaw onboard

  • I understand this is powerful and inherently risky. Continue? -> Yes
  • Onboarding mode -> QuickStart
  • Config handling -> Update values
  • Model/auth provider -> OpenAI
  • OpenAI auth method -> OpenAI Codex(chatGPT OAuth)

需要说明，如果没有自动弹出浏览器窗口，就手动复制网址到浏览器打开，输入账号登录授权，复制回调地址到 OpenClaw。

  • Default model -> openai-codex/gpt-5.3-codex
  • Select channel (QuickStart) -> Skip for now
  • Configure skills now? (recommended) -> No
  • Enable hooks? -> Skip for now
  • Gateway service already installed -> Restart
  • How do you want to hatch your bot? -> Hatch in TUI

到这一步，你的云端 AI 助手就已经跑起来了。
