Codex CLI 体验

发表于2026年2月27日 · 257个字

这几天看到很多中转站放出免费 API，体验下命令行编程方式。

1. 安装

  1. 安装 CLI: npm i -g @openai/codex
  2. 验证版本: codex --version

2. 配置

  1. 如果使用中转站则需要提前配置，参考指南。
  2. 如果使用官方账号，在首次 codex 后自动弹网页一键登录即可。

3. 使用方法

  1. 直接输入 codex 进入交互式对话
  2. 在项目目录下运行 codex，Codex 会自动识别项目上下文
  3. 输入 exit 或按 Ctrl+C 退出

到这一步，就可以愉快的编程了。
