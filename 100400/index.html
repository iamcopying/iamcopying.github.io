CS+ V4.08 版本的两个坑

发表于2024年12月18日 · 242个字

瑞萨芯片开发必看

1. 容量大小异常

以芯片型号 R5F100PG 为例，ROM 大小是 128KB，使用 V4.08 版本附带的编译器生成的 MAP 文件中 ROM SIZE = 10000H，换算下是 64KB，容量小了一半，使用 V4.07 及以下版本正常，ROM SIZE = 20000H。

2. 定时器不准

V4.08 版本 debug 时定时器不准，现象是 1 分钟定时，掐表看是 1 分零 4 秒，可能是主频偏差，没仔细查，V4.07 以下版本正常。
